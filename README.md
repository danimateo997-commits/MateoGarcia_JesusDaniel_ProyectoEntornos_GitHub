# Proyecto Entornos de Desarrollo

## Descripción
Este proyecto contiene el trabajo realizado para la asignatura de Entornos de Desarrollo que engloba el proyecto de Hector
con el codigo java que gestiona una aplicacion de Eventos. Por otro lado, tenemos el proyecto de BD de Jorge donde encontraremos
una BD SQL que gestiona pedidos de un restaurante donde enconrraemos consultas con procedimientos, funciones y Triggers

## Estructura
- **codigo-jorge/**: Proyecto de Jorge
- **codigo-hector/**: Proyecto de Hector
- **documentacion/**: Documentación complementaria
